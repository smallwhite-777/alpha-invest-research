# 综合评分标准（升级版）

## 一、三项能力指标

### 1.1 拟合分 (Fitting Score)

**定义**：是否学会了锚点报告体现的高质量研究结构、推理方式、证据组织和时间处理方式

**计算公式**：
```
拟合分 = (Q1得分 + Q7得分) / 2
```

**目标阈值**：
| 等级 | 分数 | 含义 |
|------|------|------|
| Level 1 | ≥70分 | 学会了基本报告结构 |
| Level 2 | ≥85分 | 掌握了分析范式 |

### 1.2 泛化分 (Generalization Score)

**定义**：是否能把这些能力迁移到其他公司、其他行业、其他年份的任务中

**计算公式**（差异化权重）：
```
泛化分 = 同类迁移分 × 0.5 + 跨行业迁移分 × 0.3 + 时间扰动分 × 0.2

其中：
- 同类迁移分 = (Q2得分 + Q6得分) / 2
- 跨行业迁移分 = (Q3得分 + Q4得分) / 2
- 时间扰动分 = Q5得分
```

**目标阈值**：
| 等级 | 分数 | 含义 |
|------|------|------|
| Level 3 | ≥70分 | 具备了迁移能力 |
| Level 4 | ≥85分 | 迁移能力强 |

### 1.3 过拟合风险 (Overfitting Risk)

**定义**：是否出现了行业术语泄漏、模板僵化、跨任务性能下降、错误迁移旧变量等现象

**计算公式**：
```
过拟合风险分 = 禁止词扣分 + 行业残留扣分 + 性能下降扣分

其中：
- 禁止词扣分：每个禁止词出现 -2分
- 行业残留扣分：每个行业特征词残留 -1分
- 性能下降扣分：(锚点分 - 迁移分) × 0.5，当迁移分低于锚点分>20分时触发
```

**风险等级**：
| 风险等级 | 分数范围 | 处理方式 |
|----------|----------|----------|
| 低风险 | 0-15分 | 可接受修改 |
| 中风险 | 15-30分 | 需额外审查 |
| 高风险 | >30分 | 拒绝修改 |

## 二、能力等级体系

| 等级 | 名称 | 条件 | 含义 |
|------|------|------|------|
| Level 0 | 未达标 | 拟合分<70 | 尚未掌握基本结构 |
| Level 1 | 入门 | 拟合分≥70 | 学会了基本报告结构 |
| Level 2 | 熟练 | 拟合分≥85 | 掌握了分析范式 |
| Level 3 | 迁移 | 泛化分≥70 | 具备了迁移能力 |
| Level 4 | 精通 | 泛化分≥85 | 迁移能力强 |
| Level 5 | 专家 | 拟合分≥90 且 泛化分≥90 且 过拟合风险<15 | 全面能力 |

## 四、五大评估维度

| 序号 | 维度 | 权重 | 说明 |

| 3 | 数据引用正确性 | 20% | 关键数据是否准确 |
| 4 | 行业变量正确性 | 20% | 是否正确使用目标行业特定变量 |
| 5 | 时间态准确性 | 15% | 是否正确处理时间上下文 |

## 五、结构完整性评分 (20%)

### 九大标准章节

1. 投资摘要/评级/目标价
2. 行业概况/市场规模
3. 行业周期分析
4. 公司介绍/主营业务
5. 竞争优势分析
6. 财务分析
7. 盈利预测
8. 估值分析
9. 风险提示

### 评分标准

| 等级 | 得分 | 标准 |
|------|------|------|
| 优秀 | 18-20分 | 包含全部九大章节，结构清晰完整 |
| 良好 | 14-17分 | 包含7-8个章节，结构较完整 |
| 一般 | 10-13分 | 包含5-6个章节，结构基本合理 |
| 较差 | 5-9分 | 章节缺失严重或结构混乱 |
| 差 | 0-4分 | 缺少核心章节或完全无结构 |

## 十一、综合评分计算

### 公式

```
综合得分 = 结构完整性(20) + 核心观点(25) + 数据正确性(20) + 行业变量(20) + 时间准确性(15) - 过拟合扣分
```

### 通过标准

| 等级 | 综合得分 | 说明 |
|------|----------|------|
| 通过 | ≥70分 | 具备基本的投研能力 |
| 良好 | ≥80分 | 具备较强的投研能力 |
| 优秀 | ≥90分 | 具备专业级投研能力 |

---

## 十二、修改保留条件

修改必须同时满足以下条件才能保留：

1. **拟合分提升或持平**（允许2分波动）
2. **泛化分不下降**（允许3分波动）
3. **过拟合风险可控**（≤30分）
4. **至少有一项实质提升**

# A6: 徐工机械深度报告 - 标准答案

## 一、投资摘要

**徐工机械(000425.SZ)是工程机械行业三大巨头之一，国企混改释放活力，电动化转型稳步推进，维持增持评级。**

### 核心观点

1. **起重机龙头地位稳固**：汽车起重机市场份额领先
2. **混改提升经营活力**：治理结构优化，激励机制完善
3. **国际化潜力大**：海外收入占比有望持续提升
4. **电动化布局全面**：与行业龙头同步推进

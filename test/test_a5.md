# A5: 三一重工历史时点分析(2021年) - 标准答案

## 投资时点：2021年

**三一重工(600031.SH) - 周期高点审视**

## 一、投资摘要

**2021年工程机械行业处于历史最强周期，三一重工作为行业龙头充分受益。但需关注周期见顶风险，维持增持评级但提示周期下行压力。**

## 二、行业状态分析（2021年视角）

### 市场规模

| 指标 | 2021年数据 | 历史位置 |
|------|------------|----------|
| 挖机销量 | ~34万台 | **历史新高** |
| 混凝土机械 | 高景气 | 历史高位 |
| 汽车起重机 | ~5万台 | 历史高位 |
| 行业营收 | ~9000亿 | 历史高位 |

### 周期位置判断

**当前位置：周期高点（2016-2021上行期顶部区域）**

| 周期特征 | 分析 |
|----------|------|
| 上行时长 | 2016-2021，约5年 |
| 涨幅 | 挖机销量从约6万台增至34万台 |
| 驱动因素 | 基建、房地产、设备更新周期 |
| 风险信号 | 增速逐步放缓、基数效应弱化 |

## 三、公司表现（2021年数据）

### 财务数据

| 指标 | 2021A | 同比 |
|------|-------|------|
| 营收 | 1068.7亿 | +6.8% |
| 净利润 | 120.3亿 | -22.0% |
| 毛利率 | 25.7% | -0.6ppt |
| 净利率 | 11.3% | -4.1ppt |
| ROE | 20.0% | -6.2ppt |

### 经营亮点

1. **市场份额持续提升**
   - 挖掘机械：30%（国内第一）
   - 混凝土机械：50%（全球第一）
   - 汽车起重机：30%（国内前三）

2. **国际化成效显著**
   - 海外收入占比持续提升
   - 产品出口快速增长

3. **研发投入加大**
   - 电动化布局加速
   - 智能化战略推进

## 四、盈利预测（2021年预测）

### 关键假设

1. 2022年挖机行业需求开始回落
2. 行业从增量市场转向存量更新市场
3. 电动化产品逐步放量

### 业绩预测

## 六、风险提示（2021年视角）

1. **周期下行风险**：行业销量可能回落20-30%
2. **房地产风险**：地产调控影响需求
3. **竞争加剧风险**：价格战可能影响毛利率
4. **原材料风险**：钢材价格大幅波动

## 七、投资建议

**评级：增持（周期高位提示风险）**

| 维度 | 评估 |
|------|------|
| 短期(1年) | ⚠️ 周期下行，业绩承压 |
| 中期(3年) | ✅ 电动化转型，竞争力提升 |
| 长期(5年) | ✅ 国际化空间广阔 |

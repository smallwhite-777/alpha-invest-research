# A1: 三一重工核心观点 - 标准答案

## 核心观点

**电动化是工程机械行业的主要趋势，老牌龙头优势延续，维持跑赢行业评级，目标价19.70元。**

## 四大核心论点

### 1. 电动化是工程机械行业的主要驱动力

**发展动力**：
- **经济性突出**：电动工程机械当前售价为燃油产品的2-3倍，但多余购置成本3年左右可收回，全生命周期降本效果明显
- **环保压力**：工程机械尾气排放相当于50-80辆国五小汽车，氮氧化物占机动车24.7%，颗粒物占108.9%
- **产业链优势**：借助国内成熟三电产业链，有望提升中国主机厂全球地位

**发展前景**：
- 叉车电动化率64%，高空作业平台93%，但挖掘机、装载机等主流品类仍处初始阶段
- 预计未来3-5年电动化将持续渗透

### 2. 竞争格局不会颠覆式改变

- 工程机械电动化主要由传统厂商自我革命引领，新进入者机会有限
- 老牌厂商在渠道、品牌、产能、技术上具有优势
- 三一、徐工、中联等头部企业已基本实现全谱系电动化

### 3. 三一重工电动化战略领先

- **战略定位**：2014年开启布局，目标全谱系电动化
- **研发投入**：年收入5%投向研发，研发人员超1600人
- **市场份额**：
  - 电动起重机93.5%（连续13个月第一）
  - 电动搅拌车62.0%（连续9个月第一）
  - 新能源重卡17.3%（全国第一）

### 4. 国际化是长期增长点

- 中国工程机械整机已具备全球竞争力
- 电动化有助于避开与海外厂商在内燃机和液压件上的性能差距
- 2021年、2022年海外业务保持较高增速

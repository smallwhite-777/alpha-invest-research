# A2: 恒立液压深度报告 - 标准答案

## 一、投资摘要

**恒立液压(601100.SH)是国内液压件行业龙头，受益工程机械电动化转型带来的设备升级需求，维持增持评级。**

### 核心观点

1. **液压件是工程机械核心零部件**，技术壁垒高，国产替代空间大
2. **挖机液压件市场份额持续提升**，泵阀产品已实现进口替代
3. **非标油缸下游应用广泛**，抗周期能力强
4. **研发投入持续加大**，技术实力国内领先

## 二、行业分析

### 液压件行业特点

| 特点 | 说明 |
|------|------|
| 技术壁垒高 | 液压系统被称为"工业之花"，精密加工要求极高 |
| 认证周期长 | 整机厂认证通常需要1-3年 |
| 规模效应明显 | 固定成本高，需要足够产能利用率 |
| 国产化率低 | 高端液压件长期被博世力士乐、派克汉尼汾等外资垄断 |

### 市场规模

- 全球液压件市场规模约300亿美元
- 中国液压件市场规模约600亿元人民币
- 工程机械应用占比约40%

## 三、竞争格局

### 全球格局

| 企业 | 市场份额 | 特点 |
|------|----------|------|
| 博世力士乐 | ~15% | 全球龙头，技术领先 |
| 派克汉尼汾 | ~10% | 美国龙头，工业液压强 |
| 伊顿 | ~8% | 综合液压供应商 |
| 恒立液压 | ~3% | 中国龙头，挖机液压件领先 |

### 国产替代进程

- **油缸**：已基本实现国产替代，恒立是龙头
- **泵**：从非挖机领域切入，逐步渗透挖机市场
- **阀**：技术难度最高，仍处于替代初期

## 四、公司竞争优势

### 1. 技术实力

- 研发投入占比持续提升
- 掌握高压液压缸核心制造技术
- 泵阀产品性能逐步接近外资品牌

### 2. 客户资源

- 国内挖机龙头客户覆盖率高
- 与三一、柳工等建立长期合作关系
- 售后服务响应速度快

### 3. 成本优势

- 人工成本优势明显
- 国内供应链完善
- 规模化生产降低成本

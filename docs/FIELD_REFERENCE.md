# 字段参考手册

本文档列出数据库中所有字段的详细定义。

## 标识字段

| 字段代码 | 英文名称 | 中文名称 | 说明 |
|---------|---------|---------|------|
| Stkcd | stock_code | 证券代码 | 6位股票代码 |
| ShortName | stock_name | 证券简称 | 公司简称 |
| Accper | report_date | 统计截止日期 | 报告期，YYYY-MM-DD格式 |
| Typrep | report_type | 报表类型 | A=合并报表，B=母公司报表 |
| Indcd | industry_code | 行业代码 | 证监会2012版行业分类 |
| Indnme | industry_name | 行业名称 | 证监会2012版行业名称 |
| Indcd1 | industry_code_csa | 行业代码1 | 中上协行业分类 |
| Indnme1 | industry_name_csa | 行业名称1 | 中上协行业名称 |
| Source | source | 公告来源 | 0=定期报告，1=IPO公告 |
| IfCorrect | is_corrected | 是否差错更正 | 0=否，1=是 |
| DeclareDate | correction_date | 差错更正披露日期 | 差错更正公告披露日期 |

## 财务指标字段 (F*前缀)

### F01 偿债能力指标

| 字段代码 | 中文名称 | 计算公式 |
|---------|---------|---------|
| F010101A | 流动比率 | 流动资产/流动负债 |
| F010201A | 速动比率 | (流动资产-存货)/流动负债 |
| F010401A | 现金比率 | 现金及现金等价物/流动负债 |
| F011201A | 资产负债率 | 负债合计/资产总计 |
| F010701B | 利息保障倍数A | (净利润+所得税+财务费用)/财务费用 |
| F011601A | 权益乘数 | 资产总计/所有者权益合计 |
| F011701A | 产权比率 | 负债合计/所有者权益合计 |

### F05 盈利能力指标

| 字段代码 | 中文名称 | 计算公式 |
|---------|---------|---------|
| F050201B | 总资产净利润率(ROA)B | 净利润/平均总资产 |
| F050502B | 净资产收益率(ROE)B | 净利润/平均股东权益 |
| F053301B | 营业毛利率 | (营业收入-营业成本)/营业收入 |
| F051501B | 营业净利率 | 净利润/营业收入 |
| F051401B | 营业利润率 | 营业利润/营业收入 |
| F050601B | 息税前利润(EBIT) | 净利润+所得税+财务费用 |
| F050801B | 息税折旧摊销前收入(EBITDA) | EBIT+折旧+摊销 |

### F08 发展能力指标

## 字段后缀说明

| 后缀 | 含义 |
|-----|------|
| A | 使用期末余额计算 |
| B | 使用平均余额计算 (期初+期末)/2 |
| C | TTM滚动计算 (Trailing Twelve Months) |

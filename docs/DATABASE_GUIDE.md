# A股上市公司财报数据库使用指南

## 概述

本数据库收录了1990年至2025年A股上市公司的财务数据，包括年度报告和季度报告。数据来源于CSMAR（国泰安）数据库，经过清洗和标准化处理，以Parquet格式存储，便于快速查询和分析。

## 数据范围

| 项目 | 年度数据 | 季度数据 |
|-----|---------|---------|
| 时间范围 | 1990-2024 | 1990-Q1 2025 |
| 数据量 | 约50万条记录 | 约200万条记录 |
| 公司覆盖 | 沪深北交易所全部A股上市公司 |

## 数据结构

### 年度数据 (`data/annual/`)

| 表名 | 中文名称 | 主要内容 |
|-----|---------|---------|
| balance_sheet | 资产负债表 | 资产、负债、所有者权益等154个字段 |
| income_statement | 利润表 | 营业收入、成本、利润等81个字段 |
| cash_flow_direct | 现金流量表(直接法) | 经营、投资、筹资活动现金流 |
| cash_flow_indirect | 现金流量表(间接法) | 净利润调节为经营活动现金流 |
| indicators_solvency | 偿债能力指标 | 流动比率、资产负债率等38个字段 |
| indicators_profitability | 盈利能力指标 | ROE、ROA、毛利率等81个字段 |
| indicators_operation | 经营能力指标 | 周转率、营业周期等76个字段 |
| indicators_growth | 发展能力指标 | 增长率指标66个字段 |
| indicators_per_share | 每股指标 | 每股收益、每股净资产等97个字段 |
| indicators_valuation | 相对价值指标 | 市盈率、市净率等36个字段 |
| indicators_cashflow | 现金流分析指标 | 现金含量、自由现金流等41个字段 |
| indicators_risk | 风险水平指标 | 财务杠杆、经营杠杆等12个字段 |
| indicators_dividend | 股利分配指标 | 每股股利、股利分配率等19个字段 |
| indicators_structure | 比率结构指标 | 资产结构、负债结构等44个字段 |
| indicators_disclosed | 披露财务指标 | 年报摘要关键指标16个字段 |
| company_master | 公司主表 | 公司基础信息和行业分类 |

### 季度数据 (`data/quarterly/`)

结构与年度数据相同，时间频率为季度。

## 主键字段

所有表共享以下主键字段：

| 字段代码 | 字段名称 | 说明 |
|---------|---------|------|
| Stkcd | 证券代码 | 6位股票代码，如 "000001" |
| Accper | 统计截止日期 | 报告期，格式 "YYYY-MM-DD" |
| Typrep | 报表类型 | A=合并报表，B=母公司报表 |

## 数据来源说明

- **数据供应商**: CSMAR (国泰安)
- **原始格式**: Stata DTA
- **转换格式**: Parquet (Snappy压缩)
- **数据更新**: 季度更新

## 会计准则说明

- **2007年之前**: 旧会计准则
- **2007年及之后**: 新会计准则

注意：2007年会计准则变更导致部分字段定义发生变化，进行跨年度分析时需注意。

## 行业分类

- **2012-2023年5月**: 证监会2012版行业分类 (`Indcd`, `Indnme`)
- **2023年5月之后**: 中国上市公司协会行业分类 (`Indcd1`, `Indnme1`)

## 使用建议

1. **数据查询**: 使用Python API进行查询，支持股票代码、日期范围、报表类型等过滤条件
2. **大数据分析**: Parquet格式支持列式读取，适合大规模数据分析
3. **财务分析**: 建议使用合并报表数据（Typrep='A'）

## 注意事项

1. 部分早期数据可能缺失
2. 金融行业公司财务报表格式与普通公司不同
3. 退市公司数据保留在数据库中
4. 使用前请检查数据完整性
